# 야간 작업 프롬프트 — C49: RS500 시뮬레이션 vs SS500 실차 데이터 교차 검증 리포트

> 작업 시작 전: `.claude/rules/*.md` 로딩 → `docs/프로젝트/TASK.md` 갱신 (C49 진행 등록)
> 완료 후: TASK.md 갱신 (C49 완료 이동) + 커밋

---

## 배경

C48에서 HIH-2의 DB130-48 데이터시트, CAN DBC를 분석하여
RS500 시뮬레이션 파라미터를 실차 기준으로 교정했다.

회사(GINT) 업무에서도 SS500 관련 유사 분석을 진행 중이다 (ISSUE_050 등).
이 두 작업의 데이터를 교차 비교하여 겹치는 부분과 갭을 정리한다.

**범위 제한:**
- BLACKTAN 시험 데이터는 SS500 차량이 아님 → 제외
- SOC/배터리 정합성 분석은 별도 업무 → 제외
- SS500 제품 자체에 관한 데이터만 사용 (CAN DBC, 모터 데이터시트, 제품설계, 부품 스펙)

### Step 3: 갭 분석 및 우선순위

**미반영 항목 우선순위:**
- **P1 (즉시 반영 가능)**: 데이터 확보 완료, 코드 변경 간단
- **P2 (xlsx 파싱 후 가능)**: 데이터는 있지만 바이너리 포맷
- **P3 (실차 데이터 필요)**: 실차 계측 또는 추가 시험 필요
- **신규 모듈 필요**: 기존 시뮬에 없는 기능 (분사 시스템, 배터리 SOC 모델 등)
