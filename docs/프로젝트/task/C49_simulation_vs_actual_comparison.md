## 1. 요약

C48에서 HIH-2 데이터 기반으로 RS500 시뮬레이션 파라미터를 교정했다.
본 리포트는 교정된 시뮬레이션과 SS500 실차 데이터를 항목별로 교차 비교하여,
**일치 항목, 불일치 항목, 미반영 항목, 데이터 미확보 항목**을 분류한다.

### SS500 vs DDABOT10-TA 핵심 차이

| 항목 | DDABOT10-TA | SS500 |
|------|-------------|-------|
| 유형 | 동력운반차 (4륜) | Speed Sprayer (궤도) |
| 모터 | 3.2 kW (0.8kW × 4) | **6.0 kW (3.0kW × 2)** |
| 배터리 | 48V 60Ah 리튬 | **48V 460Ah LiFePO4** |
| 최대속도 | 2.1 km/h | **4 km/h** |
| 분사 시스템 | 없음 | 펌프 + 팬 + 솔레노이드 |
| 적재 | 300 kg (적재함) | 500L 탱크 (~500 kg) |

→ SS500은 DDABOT10-TA보다 **모터 출력 1.9배, 배터리 7.7배, 최대속도 1.9배** 큰 차량.

## 8. 회사 업무 연관 (ISSUE_050 접점)

### ISSUE_050: SS500 vs 기성 SS기 성능 비교

ISSUE_050에서 비교한 항목 중 RS500 시뮬레이션과 겹치는 영역:

| ISSUE_050 항목 | RS500 시뮬 활용 가능성 | 비고 |
|-------------|-------------------|------|
| 분사 성능 (펌프 30bar/24L, 노즐 12개) | 향후 분사 모델 개발 시 입력 파라미터 | P3 |
| 팬 성능 (프로펠러×4, ~8000rpm) | 팬 에너지 소비 모델에 활용 가능 | P3 |
| 주행 성능 (미진행) | **RS500 시뮬로 사전 예측 가능** | P1 — 바로 활용 가능 |

**핵심 접점**: ISSUE_050에서 SS500 주행 성능 분석이 미진행 상태인데,
RS500 시뮬레이션으로 **직진 속도, 선회 반경, 등판 능력, 에너지 소비**를 사전 예측할 수 있다.
이 결과를 ISSUE_050 주행 성능 분석에 참고 데이터로 활용 가능.
