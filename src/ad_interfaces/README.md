# ad_interfaces

실차 CAN 통합용 커스텀 ROS2 인터페이스 패키지.

## 현재 상태

**시뮬레이션 파이프라인에서는 미사용.**
실차(SS500) CAN 연동 시 사용 예정 (STP 도면 수령 + C63 이후).

## 정의된 인터페이스

| 인터페이스 | 타입 | 용도 |
|-----------|------|------|
| `DrivingCommand.msg` | Message | 주행 명령: 목표 속도, 조향각, 브레이크, 주행 모드, 비상정지 |
| `VehicleState.msg` | Message | 차량 상태: GPS 좌표, 속도, 방향, 가속도, 기어, 상태 플래그 |
| `SetMode.srv` | Service | 모드 변경: 요청(모드명) → 응답(성공 여부, 메시지) |
